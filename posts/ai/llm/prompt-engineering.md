# Prompt 工程核心技巧

Prompt 工程是充分发挥 LLM 能力的关键。

## 核心原则

1. **清晰具体的指令** — 模糊的Prompt产出模糊的结果
2. **提供上下文和示例** — 让模型理解你的需求背景
3. **分解复杂任务** — 逐步引导比一步到位更有效
4. **迭代优化** — 根据输出持续调整Prompt

## 常用技巧

### 思维链（Chain of Thought）

使用思维链提示可以显著提高推理能力：

```
问题：计算 23 * 56
思考：先算 20*56=1120，再算 3*56=168，1120+168=1288
答案：1288
```

###Few-Shot Learning

提供几个示例让模型理解模式：

```
输入：苹果 -> 水果
输入：胡萝卜 -> 蔬菜
输入：牛肉 -> ?
```

### 结构化输出

明确要求模型以特定格式返回：

```
请以JSON格式返回：
{
  "name": "姓名",
  "age": 年龄,
  "city": "城市"
}
```

## 进阶策略

- **角色扮演**：让模型扮演特定角色回答
- **约束条件**：明确回答的范围和限制
- **追问机制**：预留追问空间
- **Temperature控制**：调整创造性vs准确性
